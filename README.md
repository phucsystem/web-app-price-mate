# PriceMate AU

Track Amazon AU product prices, view price history, and get alerts when prices drop.

## Tech Stack

- **Backend:** .NET 10 Web API (C#, LTS)
- **Frontend:** Next.js + Tamagui (React, TypeScript)
- **Database:** PostgreSQL
- **Infrastructure:** AWS CDK (TypeScript)
- **Local Dev:** Docker Compose

## Project Structure

```
docs/           # IPA documentation (SRD, UI_SPEC)
plans/          # Implementation plans and reports
prototypes/     # HTML mockup prototypes
```
